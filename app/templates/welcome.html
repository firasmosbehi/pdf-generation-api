PDF Generation API
v1.0.0

Generate production-ready PDFs from HTML, CSS, or Jinja2 templates.

This API converts raw markup or structured template data into high-quality PDF files using headless Chromium via Playwright. Include X-API-Key on /generate.

Open Swagger UI View ReDoc OpenAPI JSON Health

HTML File Mode

Upload html_file (+ optional css_file) to render directly.

Template Mode

Upload template_file and optional data_file (JSON object).

curl -X POST http://127.0.0.1:8000/generate \
  -H "X-API-Key: YOUR_API_KEY" \
  -F "html_file=@./samples/html/operations_review_q4_2025.html;type=text/html" \
  -F "css_file=@./samples/css/operations_review_q4_2025.css;type=text/css" \
  -F "filename=hello" \
  --output hello.pdf
